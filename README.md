AVISO: Esta parte del proyecto es solamente el sector del FRONTEND. La funcionalidad dentro de la pagina es erronea (hay algunos botones que no funcionan) pero se puede acceder a todas las secciones siguiendo una secuencia de pasos:
    PRESENTACION: Tocar el boton de Ya tengo cuenta: Inciar Sesion ----> LOGIN
    LOGIN: El boton ingresar a mi cuenta va al WorkerHome o ClientHome depende que se seleccione. Registrarte aqui ---> REGISTER
    REGISTER: Solo funciona el boton de "Ya tenes cuenta?"
    WorkerHome & ClientHome: Solo funciona la navbar para dirigirse al sector del chat
    CHAT: Se puede acceder a un chat en especifico o volver al sector principal

PASOS PARA PODER INICIAR EL PROYECTO

1. Tener npm instalado:
    a. Ir al cmd
    b. Introduucir "npm install"

2. Ejecutar el programa
    a. Dentro del cmd introducir "cd (directorio donde se ubique la carpeta)"
    b. Introducir "npm run dev"
    c. Entrar al localhost
